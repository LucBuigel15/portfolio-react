# portfolio-react

Portfolio gemaakt in React.

---

## Met welke technologieen heb ik deze portfolio gemaakt ?

Deze portfolio is gemaakt in **React** en **TailwindCSS**.

## Hoe run je dit project?

1. Download of clone het project: `git clone git@github.com:LucBuigel15/portfolio-react.git`
2. Open een terminal via: of de normale terminal, of de terminal die in VSCode zit.
3. Als je de terminal hebt geopend typ dan: `npm i`
4. Typ vervolgens ook in de terminal: `npm install tailwindcss @tailwindcss/vite`
5. Wanneer de dependencies zijn geinstalleerd, typ dan: `npm run dev`
